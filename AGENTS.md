# Agent Guidelines

## After Approval
- Apply changes silently.
- Do NOT repeat or paste the updated file content in the response.
- Do NOT log full file changes in chat.
- Only confirm completion briefly.

## Context Management
- Avoid large outputs after acceptance.
- Never re-send full files unless explicitly requested.

## Exceptions
- If user asks "show final file" → then output full content.
- If diff is too large → summarize + provide key hunks only.
